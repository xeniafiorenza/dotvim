## Installation

```
git clone https://github.com/xeniafiorenza/dotvim.git ~/.vim
```

Create symlinks:
```
ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/gvimrc ~/.gvimrc
```

Fetch submodules:
```
cd ~/.vim
git submodule init
git submodule update
```
